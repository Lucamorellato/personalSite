Run Forecast Run

Skills used:ES6 | RESTful API | jQuery | Pair Programming | Google Places API | Responsive | HTML5 | SASS

This is Run Forecast Run. The last stop for a runner on the way out the door. Up to date current weather based on the conditions that affect you mid-run. Behind the scenes this app fetches accurate coordinates using Google Places API and passes those through the Dark Sky API for up to the minute weather and two hour forecast. Rest Stop Reviews

Skills used:React.js | ES6 | Firebase | HTML5 | Responsive | CSS3

Rest Stop Reviews is a place to post, like and sort through user generated reviews of rest stops. Born from a love of ripping across the continent in my Subaru station wagon, the info on this website is stored on Firebase to be read or added to at any time and any place. This single page site is written in React.js and provided me with my first taste of interacting with databases. Post a review and start exploring!

Garbage Sorter

Skills used:React.js | ES6 | Pair Programming | RESTful API | Firebase Authentication | Responsive | HTML5 | CSS3 | Nivo.Rocks | React Router

Garbage Sorter is a multi-page collaborative React project. A group consisting of Em Roe, Eric Wing and myself were tasked with creating a website that allowed users to track their weekly household waste in a interactive way. Users info is stored in Firebase so they can return frequently and can track their waste production weekly and monthly. The website also features the use of the City of Toronto API so users can search 1000s of items to determine how to properly dispose of their waste.
